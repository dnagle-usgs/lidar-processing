# $Id$

This file documents the steps needed to install ALPS and its required software
on a Linux computer. In order to follow all of these instructions, you will
may need root or superuser access to the machine.

Overview:
1. Set up Directories
2. Set up Paths
3. Install ActiveTcl
4. Install rlwrap
5. Install Yorick
6. Install Yorick-Z
7. Install Yeti
8. Install ALPS
9. Install GEOID files

Set up Directories
------------------
1. Create the directories for the EAARL/ALPS system.

	sudo mkdir -p /opt/eaarl
	sudo chown `id -un`.`id -gn` /opt/eaarl
	mkdir -p /opt/eaarl/packages/src
	mkdir /opt/eaarl/bin

2. Create the directory for the files you'll be downloaded in the rest of the
document. Put all downloads in it.

	mkdir /opt/eaarl/tarfiles

Set up Paths
------------
Note: This section will have to be followed by each user who wants to run
ALPS.

1. Edit the file ~/.cshrc in your favorite editor. For example:

    gedit ~/.cshrc

It's likely that this file may be empty or not exist; that is normal. Add the
following to the end of the file, then save and exit.

    set path=(/opt/eaarl/packages/ActiveTcl/bin /opt/eaarl/bin $path)

2. Edit the file ~/.bashrc in your favorite editor. For example:

    gedit ~/.bashrc

It's likely that this file may be empty or not exist; that is normal. Add the
following to the end of the file, then save and exit.

    export PATH=/opt/eaarl/packages/ActiveTcl/bin:/opt/eaarl/bin:$PATH

ActiveTcl
---------
1. Download version 8.4.17 of ActiveTcl from their website. Select the
"-x86_64.tar.gz" file if you have a 64 bit system. Otherwise, select the
"-ix86.tar.gz" file (most people will use this one).

	http://downloads.activestate.com/ActiveTcl/Linux/8.4.17/

2. Untar the tarball as a directory in /opt/eaarl/packages/src/.

	cd /opt/eaarl/packages/src
	tar zxvf /opt/eaarl/tarfiles/ActiveTcl8.4.17-linux-ix86.tar.gz

3. Run the install script for ActiveTcl. When it prompts for an installation
   location, choose /opt/eaarl/packages/ActiveTcl8.4.17.

	cd ActiveTcl8.4.17-linux-ix86
	./install.sh

4. Add link for ActiveTcl.

    cd /opt/eaarl/packages
    ln -sf ActiveTcl8.4.17 ActiveTcl

rlwrap
------
1. Download the rlwrap from the lidar.net server.

	lidar.net:/mnt/alps/eaarl/tarfiles/rlwrap-0.21.tar.gz

2. Untar the tarball as a directory in /opt/eaarl/packages/src.

	cd /opt/eaarl/packages/src
	tar zxvf /opt/eaarl/tarfiles/rlwrap-0.21.tar.gz

3. Compile and install.

	cd rlwrap-0.21
	./configure --prefix=/opt/eaarl
	make
	make install

Yorick
------
1. Download Yorick 2.1.04 from lidar.net:

	lidar.net:/mnt/alps/eaarl/tarfiles/yorick-2.1.04.tgz

2. Untar the tarball as a directory in /opt/eaarl/packages/src/.

	cd /opt/eaarl/packages/src
	tar zxvf /opt/eaarl/tarfiles/yorick-2.1.04.tgz

3. Build yorick.

	cd yorick-2.1.04
	make relocatable

4. Install yorick.

	cd /opt/eaarl/packages
	tar zxvf /opt/eaarl/packages/src/yorick-2.1.04/yorick-2.1-04.tgz

5. Add links to yorick in /opt/eaarl/bin

	cd /opt/eaarl/bin
	ln -sf /opt/eaarl/packages/yorick-2.1-04/bin/yorick
	ln -sf /opt/eaarl/packages/yorick-2.1-04/bin/gist

Note: Before proceeding to the next steps, you may need to start a new shell
session in order for your shell to "notice" that yorick has been installed.

Yorick-Z
--------
1. Download Yorick-Z 1.2 from lidar.net:

	lidar.net:/mnt/alps/eaarl/tarfiles/yorick-z-1.2.tgz

2. Untar the tarball as a directory in /opt/eaarl/packages/src/.

	cd /opt/eaarl/packages/src
	tar zxvf /opt/eaarl/tarfiles/yorick-z-1.2.tgz

3. Compile and install.

	cd yorick-z-1.2
	./configure --yorick=/opt/eaarl/bin/yorick
	make
	make check
	make install

Yeti
----
1. Download Yeti 6.2.1 from lidar.net:

	lidar.net:/mnt/alps/eaarl/tarfiles/yeti-6.2.1.tar.gz

2. Untar the tarball as a directory in /opt/eaarl/packages/src

	cd /opt/eaarl/packages/src
	tar zxvf /opt/eaarl/tarfiles/yeti-6.2.1.tar.gz

3. Compile and install.

	cd yeti-6.2.1
	/opt/eaarl/bin/yorick -batch ./config.i
	make
	make install

ALPS
----
1. Obtain ALPS, either through CVS or as a tarball, and install as
/opt/eaarl/lidar-processing. For the tarball:

	cd /opt/eaarl
	tar zxvf /opt/eaarl/tarfiles/alps.tar.gz

2. Build and install the ALPS yorick plugins.

	cd /opt/eaarl/lidar-processing/yorick-extend/rcf
	/opt/eaarl/bin/yorick -batch make.i
	make
	make install
	make clean

GEOID Files
-----------
1. Obtain the GEOID files from lidar.net:

	lidar.net:/mnt/alps/eaarl/tarfiles/geoid03_pbdfiles.tgz

2. Untar the tarball as a directory in /opt/eaarl/lidar-processing

	cd /opt/eaarl/lidar-processing
	tar zxvf /opt/eaarl/tarfiles/geoid03_pbdfiles.tgz

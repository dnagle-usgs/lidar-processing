# $Id$

This file documents the steps needed to install ALPS and its required software
on a Linux computer. In order to follow all of these instructions, you will
need root or superuser access to the machine.

Note that the path and file names given are examples and may vary.

Overview:
1. Install ActiveTcl
2. Install rlwrap
4. Install Yorick
4. Install Yorick-Z
5. Install Yeti
6. Install Ytk
7. Install ALPS

ActiveTcl
---------
1. Download the latest version of ActiveTcl.

	http://activestate.com/Products/activetcl/

2. Untar the tarball as a directory in /usr/local/src/.

	cd /usr/local/src
	sudo tar zxvf /path/to/ActiveTcl-8.4.14-linux-ix86.tar.gz
	sudo chown -R `whoami` ActiveTcl8.4.14

3. Run the install script for ActiveTcl. When it prompts for an installation
location, choose /usr/local/ActiveTcl/.

	cd ActiveTcl8.4.14
	sudo ./install.sh

4. Add links to ActiveTcl in /usr/local/bin

	cd /usr/local/bin
	sudo ln -sf /usr/local/ActiveTcl/bin/page
	sudo ln -sf /usr/local/ActiveTcl/bin/tclsh
	sudo ln -sf /usr/local/ActiveTcl/bin/tclsh8.4
	sudo ln -sf /usr/local/ActiveTcl/bin/teacup
	sudo ln -sf /usr/local/ActiveTcl/bin/tkcon
	sudo ln -sf /usr/local/ActiveTcl/bin/wish
	sudo ln -sf /usr/local/ActiveTcl/bin/wish8.4

rlwrap
------
1. Download the latest version of rlwrap.

	http://utopia.knoware.nl/~hlub/rlwrap/

2. Untar the tarball as a directory in /usr/local/src/.

	cd /usr/local/src
	sudo tar zxvf /path/to/rlwrap-0.30.tar.gz
	sudo chown -R `whoami` rlwrap-0.30

3. Compile and install.

	cd rlwrap-0.30
	./configure
	make
	sudo make install

Yorick
------
1. Download the latest version of Yorick.

	http://sourceforge.net/project/showfiles.php?group_id=148451

2. Untar the tarball as a directory in /usr/local/src/.

	cd /usr/local/src
	sudo tar zxvf /path/to/yorick-2.1.04.tgz
	sudo chown -R `whoami` yorick-2.1.04

3. Build yorick.

	cd yorick-2.1.04
	make relocatable

4. Install yorick.

	cd /usr/local
	sudo tar zxvf /usr/local/src/yorick-2.1.04/yorick-2.1-04.tgz
	sudo chown -R `whoami` yorick-2.1-04

5. Add links to yorick in /usr/local/bin

	cd /usr/local/bin
	sudo ln -sf /usr/local/yorick-2.1-04/bin/yorick
	sudo ln -sf /usr/local/yorick-2.1-04/bin/gist

Yorick-Z
--------
1. Download the latest version of Yorick-Z.

	http://sourceforge.net/project/showfiles.php?group_id=148451

2. Untar the tarball as a directory in /usr/local/src/.

	cd /usr/local/src
	sudo tar zxvf /path/to/yorick-z-1.2.tgz
	sudo chown -R `whoami` yorick-z-1.2

3. Compile and install.

	cd yorick-z-1.2
	./configure
	make
	make check
	sudo make install

Yeti
----
1. Download the latest version of Yeti.

	http://www-obs.univ-lyon1.fr/labo/perso/eric.thiebaut/yeti_download.html

2. Untar the tarball as a directory in /usr/local/src

	cd /usr/local/src
	sudo tar jxvf /path/to/yeti-6.2.1.tar.bz2
	sudo chown -R `whoami` yeti-6.2.1

3. Compile and install.

	cd yeti-6.2.1
	yorick -batch ./config.i
	make
	sudo make install

YTK
---
1. Dowload version 1.0a of YTK (you must use exactly this version).

	https://sourceforge.net/project/showfiles.php?group_id=42998

2. Untar the tarball as a directory in /usr/local/src

	cd /usr/local/src
	sudo tar zxvf /path/to/ytk-1.0a.tgz
	sudo chown -R `whoami` ytk-1.0a

3. Copy the required files to their appropriate places.

	cd ytk-1.0a
	sudo mkdir /usr/local/yorick-2.1.04/contrib
	sudo cp ytk.i /usr/local/yorick-2.1.04/contrib/
	sudo cp ytk.gif /usr/local/yorick-2.1.04/contrib/

ALPS
----
1. Obtain ALPS, either through CVS or as a tarball, and install as
/opt/eaarl/lidar-processing. For the tarball:

	cd /opt/
	sudo mkdir eaarl
	sudo chown -R `whoami` eaarl
	cd eaarl
	tar zxvf /path/to/alps.tar.gz

2. Build and install the ALPS yorick plugins.

	cd /opt/eaarl/lidar-processing/yorick-extend/rcf
	yorick -batch make.i
	make
	sudo make install
	make clean
